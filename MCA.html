2.computer architecture

  1. Cache: Cache is a type of memory that is located between the processor and main memory. It is used to temporarily store frequently accessed data and instructions, in order to improve system performance.
  2. Processor: Also known as the Central Processing Unit (CPU), the processor is the main component of a computer system. It executes instructions and manages the flow of data within the system.
  3. Memory: There are several types of memory in a computer system, including Random Access Memory (RAM) and Read-Only Memory (ROM). RAM is used to temporarily store data and program instructions, while ROM stores firmware and other permanent data.
  4. Input/Output (I/O): This refers to the various devices that allow users to interact with a computer system, such as keyboards, mice, displays, and printers. I/O devices communicate with the processor through a set of standard interfaces.
  5. Instruction Set Architecture (ISA): This is the interface between the software and hardware of a computer system. It defines the instructions that the hardware can execute, as well as the format of those instructions.
